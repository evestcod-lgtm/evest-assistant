# Офлайн-модель распознавания речи Vosk (нужна перед сборкой)

Сюда нужно распаковать содержимое маленькой русской модели Vosk.

## Как получить

1. Скачайте `vosk-model-small-ru-0.22.zip` (~45 МБ) отсюда:
   https://alphacephei.com/vosk/models
   (ищите модель **vosk-model-small-ru-0.22**, "small" — специально
   компактная версия для мобильных устройств).
2. Распакуйте архив.
3. Содержимое папки `vosk-model-small-ru-0.22/` (файлы `am/`, `conf/`,
   `graph/`, `ivector/` и т.д.) скопируйте **прямо сюда**, в
   `app/src/main/assets/vosk-model/`, БЕЗ вложенной папки — то есть
   `app/src/main/assets/vosk-model/am/...`, а не
   `app/src/main/assets/vosk-model/vosk-model-small-ru-0.22/am/...`.

## Автоматически через GitHub Actions

Наш workflow (`.github/workflows/build.yml`) уже умеет сам скачивать и
распаковывать эту модель перед сборкой — никаких ручных действий с телефона
не требуется. Просто запустите workflow в GitHub Actions (вкладка Actions →
Run workflow), и он всё сделает сам.

## Если не сделать этот шаг вручную и не использовать workflow

Приложение соберётся, но распознавание речи после активации не сработает —
в интерфейсе появится сообщение "офлайн-модель распознавания не найдена".
Все остальные функции (открытие приложений, TTS, Groq-диалог через ручной
ввод) не пострадают.
